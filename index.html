Installation

Up and running
in minutes

macOS only. Requires Walmart network or VPN.

1

Run the one-liner installer

Open Terminal and paste. Installs Wibey CLI if missing, clones sArthI, and configures all 12 MCP servers.

bash <(curl -fsSL https://gecgithub01.walmart.com/raw/WITDnA/sarthi/main/install.sh)
2

Complete the auth wizard

6 browser windows open — one per service. Log in with your Walmart credentials. Takes ~10 minutes. Never repeated.

bash ~/sarthi/scripts/first-time-auth.sh
3

Restart Wibey and verify

Close the terminal and relaunch Wibey. Run the health check to confirm all systems are connected.

wibey → /sar-setup
