Installation

One command.
Everything configured.

macOS only. Requires Walmart network or VPN. Uses your existing git credentials — no token flags needed.

1

Run the installer

Open Terminal and paste. Installs Wibey CLI if missing, clones sArthI, configures all MCP servers, and walks you through 6 one-time auth logins — all automatically.

git clone https://gecgithub01.walmart.com/WITDnA/sarthi.git ~/sarthi && bash ~/sarthi/install.sh

6 browser windows will open for Walmart SSO — one per service. Complete each login and they close automatically. This happens once and is never repeated.

2

Restart Wibey

Close the terminal and relaunch Wibey. The new MCP servers and skills load automatically.

wibey
3

Start sArthI

Type /custom/sarthi inside Wibey. On first run it checks everything automatically. Run /custom/sarthi setup anytime to re-verify connections.

/custom/sarthi

Your entire stack,

sArthI doesn't just diagnose — it clears failed tasks, triggers reruns, and restarts services autonomously. Self-healing, not just alerting.

Query BigQuery, browse GCS, inspect Dataproc, trace Hudi timelines, and investigate Azure, CosmosDB, and Redis — all from one place.

Kubernetes / WCNP

Inspect pods, stream logs, scale deployments, and restart rollouts across 30+ clusters — without kubectl.

ServiceNow & Access Management

Draft CRQs, triage incidents, approve or reject RITMs, and raise AD group access requests — all from chat.

Inbox, Teams & Slack

Read and reply to Outlook emails, monitor Teams channels, and post to Slack — without leaving Wibey.

Cost Intelligence
