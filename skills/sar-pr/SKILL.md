# sar-pr — Pull Request Creator

## Purpose
sArthI's git hands. Once a fix is proposed or a scaffold is generated, this skill
handles the entire git workflow: branch, commit, push, PR — with a properly
formatted description referencing Jira and the resolution summary.

## Auth
Uses `~/.gitconfig` PAT token for GEC GitHub (`gecgithub01.walmart.com`).
No extra setup needed if token is configured.

## GEC GitHub repo
Set in `~/.wibey/knowledge/environments.json → github.org` and `github.base_url`.
